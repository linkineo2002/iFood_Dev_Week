# Restaurante API

Sacola API para Servir uma Aplicação com Domínio de Delivery

### Status do Projeto

Sacola API.

### Features

-  Incluir itens na sacola
-  Visualizar sacola
-  Fechar sacola
-  Excluir item da sacola
